# UI コンテキスト (UI Context)

## テーマ (Theme)

[全体的なビジュアル言語を記述する — 例: ダークのみ。
ライトモードなし。デザイン言語はダークテクニカルな
ワークスペース — ほぼ黒の背景、レイヤー化されたサーフェス、
インタラクティブ要素には鮮やかなアクセントカラーを使用する。]

## カラー (Colors)

[カラートークンを CSS カスタムプロパティとして定義する。
すべてのコンポーネントはこれらのトークンを使用しなければならない —
ハードコードされた hex 値は禁止。]

| 役割             | CSS 変数            | 値        |
| ---------------- | ------------------- | --------- |
| ページ背景       | `--bg-base`         | `#[hex]`  |
| サーフェス       | `--bg-surface`      | `#[hex]`  |
| 主要テキスト     | `--text-primary`    | `#[hex]`  |
| 控えめなテキスト | `--text-muted`      | `#[hex]`  |
| 主要アクセント   | `--accent-primary`  | `#[hex]`  |
| ボーダー         | `--border-default`  | `#[hex]`  |
| エラー           | `--state-error`     | `#[hex]`  |
| 成功             | `--state-success`   | `#[hex]`  |

## タイポグラフィ (Typography)

| 役割              | フォント           | 変数          |
| ----------------- | ------------------ | ------------- |
| UI テキスト       | [例: Geist Sans]   | `--font-sans` |
| コード/等幅       | [例: Geist Mono]   | `--font-mono` |

## ボーダー半径 (Border Radius)

| コンテキスト       | クラス           |
| ------------------ | ---------------- |
| インライン/小UI    | `rounded-[size]` |
| カード/パネル      | `rounded-[size]` |
| モーダル/オーバーレイ | `rounded-[size]` |

## コンポーネントライブラリ (Component Library)

[例: Tailwind の上に shadcn/ui を採用。コンポーネントは
components/ui/ に配置する。新しいコンポーネントは手書きせず
CLI を使って追加する。]

- [パターン — 例: エディタ: ビューポート全体を
  左サイドバー / 中央キャンバス / 右サイドバーで分割]
- [パターン — 例: サイドバー: 固定幅でボーダーセパレータ付き]
